# Portfolio

Web personal de portfolio, hecha con HTML, CSS y JavaScript puro (sin frameworks ni build step).

## Estructura

- `index.html` — página principal con todas las secciones
- `css/style.css` — estilos
- `js/main.js` — menú móvil y pequeñas interacciones
- `assets/` — imágenes y otros recursos

## Cómo verla en local

Abre `index.html` directamente en el navegador, o sirve la carpeta con cualquier servidor estático (por ejemplo, la extensión "Live Server" de VSCode).

## Cómo publicarla con GitHub Pages

1. Sube este repositorio a GitHub.
2. Ve a **Settings → Pages**.
3. En "Build and deployment", selecciona la rama `main` y la carpeta `/ (root)`.
4. Guarda. En unos minutos la web estará disponible en `https://<tu-usuario>.github.io/<nombre-del-repo>/`.
